# Cistanthelongiscapa_2026
Análisis transcriptómico de Cistanthe Longiscapa año 2026

DEGs14dias_up.csv = DEGs upregulated en tratamiento de sequía 14 días vs control 14 dias, obtenidos en análisis de expresión diferencial en librerias de laboratorio

Matriz_condiciones_laboratorio_VST.csv = matriz de conteo de laboratorio transformada por VST

Matriz_sitios_tiempos.csv = matriz de conteo de muestras de sitio + tiempo transformada por VST

Diccionario_funcional.csv = Archivo de homología de CL con AT11 y descripción funncional asignada por eggnogMapper

Se incluyen los reportes de multiQC de los tres conjuntos de librerias (Laboratorio, Sitios, 4 tiempos) que incluyen: secuencias crudas, secuencias trimmeadas por Trim galor!, Mapeo a genoma de CL mediante STAR, asignación de transcritos mediante featureCounts.
